# Funding Collective

Collective is a community-driven platform built to enable meaningful collective action. Like most open-source projects, we need community support to maintain and grow the platform.

## Why We Need Funding

- **Hosting Costs**: Vercel Pro for deployments and edge functions
- **Backend Services**: Supabase Pro for database, authentication, and storage
- **Development**: Server time for testing, CI/CD, and monitoring
- **Community Support**: Documentation, community management, and user support

## Funding Platforms

Choose your preferred way to support us:

- [GitHub Sponsors](https://github.com/sponsors/[your-username]) - Direct monthly support
- [Open Collective](https://opencollective.com/collective) - Transparent community funding
- [Patreon](https://patreon.com/collective) - Creator-focused support
- [Ko-fi](https://ko-fi.com/collective) - One-time or monthly support

## Cost Transparency

We believe in full transparency about how your contributions are used:

- **Hosting (40%)**: Vercel, domain, monitoring
- **Backend Services (35%)**: Supabase Pro, storage, API limits
- **Development Tools (15%)**: CI/CD, testing, security
- **Community Support (10%)**: Documentation, moderation, support

## Budget Breakdown

### Monthly Costs (with 1000 active users)
- Vercel Pro: $20/month
- Supabase Pro: $25/month
- Domain & monitoring: $10/month
- **Total: ~$55/month**

## How to Use Funding

All funding goes directly into:
1. **Infrastructure**: Keeping the platform running reliably
2. **Development**: Building new features and improving user experience
3. **Security**: Maintaining high security standards
4. **Community**: Supporting users and contributors
